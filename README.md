# Samurai Revanche Multiplayer ⚔️

Jogo de adivinhação para 2 jogadores com modo revanche infinito.

## 🎮 Como jogar
1. Player 1 escolhe número secreto de 1 a 100
2. Player 2 tenta adivinhar com dicas de "maior" ou "menor"
3. Ao acertar, Player 2 pode pedir revanche e os papéis invertem
4. Repete até alguém negar a revanche

## 🧠 O que aprendi nesse projeto
- Funções e escopo com `global`
- Loops `while` aninhados para controle de fluxo
- Validação de input com `.lower().strip()`
- Manipulação de listas com `append()` e `len()`
- Debug de erros de indentação e métodos sem `()`
- Tuple unpacking: `a, b = b, a` pra inverter jogadores
- Desenvolvimento 100% no Android usando Acode
